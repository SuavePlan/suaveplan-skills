# @suaveplan/types

Types-only foundation package. Zero runtime code, zero dependencies, zero bundle bytes. Provides the canonical shared interfaces that allow every `@suaveplan/*` package to accept and return the same shapes without coupling to a concrete implementation: `Clock` (`() => number`), `IdGenerator` (`() => string`), `Rng` (seeded random interface), `MessageResolver` (i18n hook), `EventBusLike<TMap>` (minimal event-bus contract), `LoggerLike` (minimal structured-logger contract), and three composable option bundles (`CancellableOptions`, `TraceableOptions`, `DeterministicOptions`). Always `import type` from this package — it emits nothing at runtime. This is the only `@suaveplan/*` dependency the testing-infrastructure packages (`testing`, `testing-dom`, `testing-e2e`) are permitted to take, because its build emits no runtime bytes and therefore creates no circular hazard.
